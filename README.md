# CodTech Data Analysis Internship

This repository contains all tasks completed during the CodTech Data Analysis Internship.

## Tasks Completed

### Task 1: Big Data Analysis Using PySpark
- Analyzed Walmart sales dataset using Apache Spark (PySpark)
- Performed data exploration and business insights extraction
- Visualized sales trends and store performance

### Task 2: Employee Attrition Prediction
- Built machine learning models to predict employee attrition
- Applied Logistic Regression, Decision Tree, and Random Forest
- Compared model performance and accuracy

### Task 3: Interactive Dashboard Development
- Developed an interactive Power BI dashboard
- Created KPI cards, sales trend analysis, and store-level filtering
- Generated business insights using visual analytics

### Task 4: Sentiment Analysis
- Performed sentiment analysis on IMDb movie reviews
- Applied TF-IDF vectorization and Logistic Regression
- Achieved approximately 89% classification accuracy

## Technologies Used

- Python
- PySpark
- Pandas
- Scikit-learn
- Power BI
- Matplotlib
- Machine Learning
