# Big Data Analysis Using PySpark

## CodTech Data Analysis Internship - Task 1

### Objective
Analyze Walmart sales data using Apache Spark (PySpark) and extract meaningful business insights from large-scale data.

### Tools Used
- Python
- PySpark
- Pandas
- Matplotlib
- Walmart Sales Dataset

### Dataset Information
The dataset contains:
- Store
- Date
- Weekly Sales
- Holiday Flag
- Temperature
- Fuel Price
- CPI
- Unemployment

Total Records: 6435

### Analysis Performed
- Loaded data using Spark Session
- Explored dataset schema
- Calculated total sales by store
- Identified top-performing stores
- Computed average weekly sales
- Compared holiday and non-holiday sales
- Visualized sales trends using charts

### Key Findings
- Store 20 generated the highest total sales.
- Average weekly sales were approximately 1.05 million.
- Holiday weeks showed higher average sales.
- Sales performance varied significantly across stores.
