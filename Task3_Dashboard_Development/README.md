# Interactive Dashboard Development Using Power BI

## CodTech Data Analysis Internship - Task 3

### Objective
Develop an interactive dashboard using Power BI to analyze Walmart sales data and provide meaningful business insights.

### Tools Used
- Power BI Desktop
- Walmart Sales Dataset

### Dashboard Features
- Total Sales KPI
- Average Weekly Sales KPI
- Total Stores KPI
- Top 10 Stores by Sales
- Holiday vs Non-Holiday Sales Comparison
- Sales Trend Analysis
- Interactive Store Filter (Slicer)

### Key Insights
- Store 20 generated the highest sales.
- Holiday weeks have higher average sales compared to non-holiday weeks.
- Sales fluctuate significantly throughout the year.
- Interactive filtering enables store-level analysis.

The dashboard provides an interactive visualization of Walmart sales performance and supports data-driven business decisions.
