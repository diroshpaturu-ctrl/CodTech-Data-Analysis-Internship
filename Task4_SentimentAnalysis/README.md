# Sentiment Analysis on IMDb Movie Reviews Using NLP

## CodTech Data Analysis Internship - Task 4

### Objective
To perform sentiment analysis on IMDb movie reviews using Natural Language Processing (NLP) techniques and machine learning models.

### Tools and Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn
- NLTK
- Scikit-Learn

### Dataset
IMDb Movie Reviews Dataset

- Total Reviews: 50,000
- Positive Reviews: 25,000
- Negative Reviews: 25,000

### Project Workflow

1. Data Loading
2. Data Exploration
3. Text Cleaning
4. Stopword Removal
5. TF-IDF Vectorization
6. Train-Test Split
7. Logistic Regression Model Training
8. Model Evaluation
9. Visualization
10. Insights and Conclusion

### Model Performance

| Metric | Value |
|----------|---------|
| Accuracy | 89.22% |
| Precision | 89% |
| Recall | 89% |
| F1-Score | 89% |

### Key Insights

- Dataset was perfectly balanced.
- No missing values were found.
- NLP preprocessing improved text quality.
- TF-IDF effectively transformed textual data into numerical features.
- Logistic Regression achieved strong classification performance.

### Conclusion

The project successfully classified movie reviews into positive and negative sentiments using NLP techniques and machine learning with an accuracy of 89.22%.
